# 작업 완료 후 자가 점검 훅 (Post-Run Self-Check Hook)

이 컨텐츠는 AI 에이전트가 하나의 '단일 태스크(3_checklist.md 항목)'를 완료했다고 판단했을 때, 루프를 종료하기 직전 **스스로 수행해야 하는 최종 퀄리티 게이트**입니다.

## ✅ 셀프 체크리스트
1. **에러 및 회귀 테스트 점검:**
   방금 작성하거나 수정한 코드가 문법 에러를 뱉지는 않는가? 기존 기능을 망가뜨리지 않았음을 확신하는가?
   > 💡 확신할 수 없다면, `skills/sprint_qa.md` 매뉴얼을 열고 스스로 터미널에서 스크립트를 컴파일/단위 테스트하여 에러 로그를 확인한 후 수정해.

2. **보안 취약점 및 하드코딩 점검:**
   API Key, 비밀번호, 중요한 인증 토큰 등을 코드에 하드코딩하지 않았는가? (`.env`를 사용하도록 안내했는가?)

3. **외부 기억 장치 동기화:**
   - [ ] 방금 해결한 작업은 `context/3_checklist.md`에 `[x]` 로 마킹했는가?
   - [ ] 다음 루프의 AI 에이전트가 변수 상태나 맥락을 잃어버리지 않도록 `context/2_context_note.md`에 모든 의도와 이슈를 기록했는가?
   - [ ] 전체 계획이 수정되었다면 `context/1_plan.md`를 업데이트 했는가?

모두 완벽하다면 비로소 이 턴을 멈추고 `LOOP_DONE` 을 선언해라.
